───────────────────────────────────────────────────────────────────────────
WHAT THIS CALCULATES
───────────────────────────────────────────────────────────────────────────

For your training configuration:
  python train_yolov12_dino.py \
      --yolo-size s \
      --dino-variant vitb16 \
      --dinoversion 3 \
      --integration single \
      --epochs 700 \
      --batch-size 2

You'll get:
  ✅ GFLOPs:       Computational complexity
  ✅ Parameters:   Total, trainable, frozen counts
  ✅ Model Size:   Memory requirements
  ✅ Performance:  Estimated inference speed

───────────────────────────────────────────────────────────────────────────
COMMON COMMANDS
───────────────────────────────────────────────────────────────────────────

All commands run from Floptest folder:

# Quick start (pre-configured)
./run_flopcal.sh

# Quick test
./test_flopcal.sh

# Base YOLOv12
python flopcal.py --yolo-size s

# Your config
python flopcal.py --yolo-size s --dino-variant vitb16 \
                  --dinoversion 3 --integration single

# With custom report name
python flopcal.py --yolo-size s --dino-variant vitb16 \
                  --dinoversion 3 --integration single \
                  --save-report my_analysis.md

# Compare sizes
python flopcal.py --yolo-size n  # nano
python flopcal.py --yolo-size s  # small (yours)
python flopcal.py --yolo-size l  # large

# Help
python flopcal.py --help

Problem: Need help
Fix: cat START_HERE.txt or python flopcal.py --help
